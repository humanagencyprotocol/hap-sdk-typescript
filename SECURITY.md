# Security Policy

## Reporting a Vulnerability

If you discover a vulnerability, please email **security@humanagencyprotocol.org** with:

- Description of the issue and potential impact
- Steps to reproduce (or proof-of-concept)
- Suggested mitigation if available

We will acknowledge receipt within 5 business days and work with you on coordinated disclosure.

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.1.x   | ✅ Security fixes   |

Older versions should be upgraded to the latest release to receive patches.

## Security Expectations

- Do not open public issues for unpatched vulnerabilities.
- Never include real API keys or user content in reports or test cases.
- Follow the Stop→Ask→Proceed philosophy when handling sensitive context.
